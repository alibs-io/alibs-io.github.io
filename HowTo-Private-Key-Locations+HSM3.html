Before You Begin

  1. Ensure that the following pre-conditions exist before configuring the nShield Connect.
    1. The Gateway appliance must not have an internal HSM. You cannot connect an nShield Connect appliance to a Gateway appliance that already uses an internal HSM.
      1. Como validar ?</b></font>
    2. Verificar a versão do ssg-nshieldpci 
       rpm -qa | grep ssg-nshieldpci
    3. Verificar se usuários layer7 e gateway estão no grupo nfast 
       egrep -i "^nfast" /etc/group

      ==» nfast: x:504:gateway,layer7

    4. Se algun dos usuários não estiver no grupo 
       usermod -g layer7 -G 'layer7,nfast' layer7
 usermod -g gateway -G 'gateway,pkcs11,nfast' gateway
